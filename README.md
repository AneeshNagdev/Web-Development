# Web Development Learning Journey

Welcome to my Web Development repository! This folder contains a comprehensive collection of practical labs, projects, and activities that I completed while learning full-stack web development. Through these exercises, I have built a strong foundation in both frontend and backend technologies.

## 🚀 Overview

This repository is structured as a series of progressive lab exercises, starting from basic HTML/CSS and advancing to server-side scripting. Each folder represents a distinct practical assignment focusing on specific web concepts. 

**Note: Each lab folder contains its own `README.md` file detailing the specific activities covered in that lab, along with a link to the live demo.**

## 📁 Repository Structure

### Frontend Fundamentals
- **`lab01` - `lab03`**: Introduction to markup and styling. Covers basic HTML elements, page structuring, and styling techniques using CSS. Includes exercises on creating responsive layouts using Grid and Flexbox.
- **`lab04` - `lab05`**: Introduction to JavaScript. Focuses on client-side interactivity, DOM manipulation, form validation, and event handling.
- **`lab06` - `lab07`**: Advanced HTML/CSS layouts, integrating media (video/images) and custom styling.

### Server-Side & Backend 
- **`lab08`**: Introduction to PHP. Covers server-side web structure and dynamic page generation using PHP.
- **`lab10` & `cgi-bin`**: Practical implementations of Common Gateway Interface (CGI) scripts.
  - Features backend scripts written in **Perl** (`lab07a.cgi`, `lab07c.cgi`) and **Python** (`lab10b.cgi`, `lab10c.cgi`) to handle form data processing and dynamic server-rendered UI.

## 🛠️ Technologies & Languages Used

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: PHP, Python, Perl, CGI (Common Gateway Interface)

## 💡 What I Learned

Throughout these labs, I gained hands-on experience in:
- Structuring semantic and accessible web pages.
- Applying creative and responsive design principles.
- Creating interactive features and manipulating the DOM using JavaScript.
- Writing backend scripts to process HTTP requests and handle form data dynamically.
