# Lab 05
 
This lab involves creating an interactive webpage using HTML, CSS, JavaScript, and jQuery. 

It features user input validation for a form (Name, Address, and Phone Number), a real-time updating digital clock, and interactive image effects. Clicking the cat image scales it up and adds a curved text effect using the Arctext.js plugin. 

**Live Demo:** [https://www.cs.torontomu.ca/~anagdev/lab5/lab05.html](https://www.cs.torontomu.ca/~anagdev/lab5/lab05.html)
