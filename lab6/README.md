# Lab 06
 
This lab involves creating vector graphics using SVG (Scalable Vector Graphics) directly within HTML.

It features basic shapes (circles, polygons) and complex custom paths using quadratic and cubic bezier curves to draw playing card suits (clubs, diamonds, hearts, and spades). It also includes SVG text styling and positioning with various fonts, strokes, and transformations.

**Live Demo:** [https://www.cs.torontomu.ca/~anagdev/lab6/lab06.html](https://www.cs.torontomu.ca/~anagdev/lab6/lab06.html)
