# Lab 04

This lab involves creating a webpage that features a BookMarks List and a Palindrome Checker, implemented using HTML and JavaScript. 

The **BookMarks List** dynamically displays a list of websites, indicating whether each link is secure (HTTPS) or insecure (HTTP) using lock icons. 

The **Palindrome Checker** evaluates various strings to determine if they are palindromes, ignoring spaces and punctuation. It includes a predefined list of test strings and an interactive text box that allows users to test their own inputs.
