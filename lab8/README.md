# Lab 08

This lab introduces server-side scripting using PHP. 

It features dynamic content generation based on the time of day (changing greetings and backgrounds), a user-interactive form that generates a customized multiplication table, and the use of cookies to save and display a user's favorite image selection across sessions.

**Live Demo:** [Lab 08 - PHP Server-Side Scripting](https://www.cs.torontomu.ca/~anagdev/lab8/lab08.php)
