# Lab 10

This lab explores both server-side form handling and modern CSS frameworks, split into two separate problems.

## Problem 1: Server-Side Form Handling
This part features an HTML form that collects information about a city (Name, Country, Image URL, and a Hex Color). The form demonstrates how the same client-side data can be submitted to and processed by different server-side scripts (ASP, Perl CGI, and Python CGI) using HTML5 `formaction` attributes on multiple submit buttons.

**Live Demo:** [Problem 1 - City Info Form](https://www.cs.torontomu.ca/~anagdev/lab10/lab10.html)

## Problem 2: Tailwind CSS Project
This part demonstrates the use of Tailwind CSS. It is a single-page project styled entirely using Tailwind utility classes via a CDN link. The page also documents the development process, including how Tailwind was installed, the approach used to build the page layout, and the difficulties faced (such as finding the correct CDN link and learning the utility classes) along with their solutions.

**Live Demo:** [Problem 2 - Tailwind Project](https://aneeshnagdev.github.io/CPS530/lab10Problem2.html)
