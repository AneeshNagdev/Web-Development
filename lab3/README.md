# Lab 03

This lab focuses on building a webpage for "Global Pet Adoption Services," a platform for adopting a diverse range of animals including cats, giraffes, and marine life. It was created using HTML and CSS to practice structuring content and styling information logically. The project features navigation menus, categorized adoption sections, organized pet profile data, and semantic structural elements to deliver a clean and intuitive user experience.
