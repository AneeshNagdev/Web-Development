# Lab 01

This lab is a simple introductory webpage dedicated to Kamet Mountain, the second-highest peak in the Garhwal region of Uttarakhand, India. It demonstrates the use of foundational HTML elements, including headings, paragraphs, styled divisions, lists, images, and embedded video.
