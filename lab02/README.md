# Lab 02

This lab is a personal biography webpage. It was created using HTML and CSS to practice styling and structuring webpage content. It features a custom layout, imported typography, icons from Google Material Symbols, image handling, and proper semantic HTML sections to organize information such as personal story, hobbies, and goals.
